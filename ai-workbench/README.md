# ai-workbench

A long-term personal project aimed at creating a maintainable, modular, and resilient personal AI operating system.

## Core Philosophy
- AI providers are temporary. The project state is permanent.
- AI tools are workers. The workbench is the brain.
- Create uninterrupted workflows that can survive rate limits, context loss, provider outages, switching tools, and changing technologies.
- Never tightly couple the system to a specific AI. Providers should be replaceable.

## Anti-Goals
- NOT a provider abuse system.
- NOT an automated restriction circumvention tool.
- NOT a browser hacking tool.
- NOT a token farming system.
- NOT about building giant autonomous agents immediately.
- NOT optimizing for maximum free tokens. Optimizing for maximum uninterrupted flow.

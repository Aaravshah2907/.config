### 🖥️ iTerm2 Karabiner Shortcut Ideas

  Quick Command Launchers (your most-used commands):

  • Hyper + Y → Launch  yazi  (file manager) in current pane
  • Hyper + G → Launch  lazygit  in current pane
  • Hyper + V → Launch  fv  (yazi with cd-on-exit)

  Pane/Tab Management:

  • Application + T → New tab (send  ⌘T )
  • Application + W → Close current pane (send  ⌘W )
  • Application + D → Split pane vertically (send  ⌘D )
  • Application + Shift+D → Split pane horizontally (send  ⌘⇧D )

  Session Quick Actions:

  • Hyper + E →  clear && exit  (your  ce  alias — instant close)
  • Hyper + R →  skreload  (reload skhd — you run this constantly)

  Service Management (frequent brew services pattern):

  • Application + S →  brew services restart skhd  (you restart this a lot)
  • Application + Y →  yabai --restart-service

  Dev Workflow:

  • Hyper + ; → Open SSH / remote session (if you use one)
  • Application + H →  cht.sh  (your cheatsheet / help alias)

  Process Control (you frequently  pkill ):

  • Application + K → Kill last backgrounded process /  pkill -f vlc
